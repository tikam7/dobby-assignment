Dobby Ads Assignment

A simple backend project built using Node.js, Express.js, and MongoDB to simulate ad/content upload handling with configurable byte sizes.

Features
Upload and store data in MongoDB
Adjustable byte size for testing (e.g. 200B, 800B, etc.)
REST API using Express.js
MongoDB integration using Mongoose
Simple and clean project structure
JSON-based API responses
Tech Stack
Node.js
Express.js
MongoDB
Mongoose

project/

Installation
git clone <repository-url>
cd project
npm install
Environment Variables

Create a .env file:

MONGO_URI=your_mongodb_connection_string
PORT=5000
Run the Project
npm start

or

node server.js
API Functionality
Create Data

Stores generated byte data into MongoDB.

Example:

{
  "size": 200
}

The size value can be changed dynamically for testing different payload sizes.
